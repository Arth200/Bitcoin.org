What Happened

A Bitcoin miner running version 0.8.0 created a large block (height 225,430) that is incompatible with earlier versions of Bitcoin.

This resulted in a blockchain fork, with miners, merchants, and users on the new version accepting and building on that block, while those on older versions rejected it and created their own blockchain.

What Is Being Done

Large mining pools using version 0.8.0 have been asked to revert to version 0.7 to establish a single blockchain compatible with all Bitcoin software.

Questions & Answers

I'm Not a Miner or a Merchant; What Should I Do?

Nothing. Your Bitcoin software will automatically switch to the correct chain, regardless of the version you are using.

Are My Bitcoins Safe?

Yes, your bitcoins are safe.

What Will Be Done

Core developers have investigated the issues that caused older versions to reject the new blocks and have released version 0.8.1 to prevent the creation of incompatible blocks. A full post-mortem document has been published.
